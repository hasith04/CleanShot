# CleanShot 📸

Material You screenshot organizer app built with Kotlin and Jetpack Compose.

## Features

* Real screenshot detection
* Storage analytics
* Material You inspired UI
* Screenshot gallery preview
* Android 14 support
* Built using Jetpack Compose

## Tech Stack

* Kotlin
* Jetpack Compose
* Material 3
* MediaStore API
* Coil Image Loading
